# Uttunen

Uttunen is an advanced calculator built with latest hot technologies

## Installation

Use the bun runtime [bun](https://bun.sh/) to install all dependecies.

```bash
bun install
```

## Getting started

```bash
// start in production mode
$ bun start

// start in development mode
$ bun dev

// run tests
$ bun test
```

## Technologies

- Bun
- Typescript <3
- Elysia
- HTMX
- Docker
